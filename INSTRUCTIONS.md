# Git & Project Workflow Cheat Sheet

Αυτός είναι ο "τυφλοσούρτης" σου για να δουλεύεις μεταξύ iMac και MacBook Air χωρίς να χάνεις δουλειά.

## 1. Όταν πας στο MacBook Air (Πρώτη φορά)

Επειδή μόλις φτιάξαμε το Git στο iMac, το MacBook Air δεν "ξέρει" ακόμα τη σύνδεση.
Το πιο ασφαλές είναι να το κατεβάσεις από την αρχή **ως νέο φάκελο**.

1.  Άνοιξε το **Terminal**.
2.  Πήγαινε στον φάκελο που βάζεις τα projects σου (π.χ. `cd Documents`).
3.  Τρέξε την εντολή:
    ```bash
    git clone https://github.com/kadamianos2711/dietitian-app.git
    ```
4.  Θα δημιουργηθεί ένας φάκελος `dietitian-app`. Μπες μέσα:
    ```bash
    cd dietitian-app
    ```
5.  Εγκατάστησε τα απαραίτητα (μια φορά χρειάζεται):
    ```bash
    npm install
    ```

---

## 2. Καθημερινή Ρουτίνα (Σε κάθε υπολογιστή)

### Α. Όταν ΞΕΚΙΝΑΣ δουλειά (START):
Πριν γράψεις οτιδήποτε, φέρνουμε τις τελευταίες αλλαγές από το cloud.

1.  Άνοιξε Terminal στον φάκελο του project.
2.  Τρέξε:
    ```bash
    git pull
    ```
3.  Ξεκίνα την εφαρμογή:
    ```bash
    npm run dev
    ```

### Β. Όταν ΤΕΛΕΙΩΝΕΙΣ δουλειά (STOP):
Στέλνουμε τη δουλειά στο cloud για να την βρούμε στον άλλο υπολογιστή.

1.  Σταμάτα τον server (Ctrl + C).
2.  Τρέξε αυτές τις 3 εντολές:
    ```bash
    git add .
    git commit -m "Douleia simera"
    git push
    ```

---

## Τι να λες σε εμένα (στο AI)
Όταν ανοίγεις νέα συνομιλία στον άλλο υπολογιστή, απλά pes μου:
> *"Γεια, είμαι στο MacBook Air. Έκανα git pull. Θέλω να συνεχίσουμε την δουλειά στο dietitian app..."*

Έτσι θα καταλάβω αμέσως πού βρισκόμαστε!
